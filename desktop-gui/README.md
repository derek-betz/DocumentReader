# Desktop GUI

PyQt5-based desktop application for Roadway-Doc-Engine.

## Installation

Install PyQt5:

```bash
pip install PyQt5
```

## Running

```bash
python desktop-gui/main.py
```

Or if installed as a package:

```bash
python -m desktop_gui.main
```

## Features

- Drag-and-drop file support
- Real-time processing progress
- INDOT sheet type identification
- Measurement and annotation extraction
- Results export to JSON or text

## Requirements

- Python 3.8+
- PyQt5
- document-reader package
